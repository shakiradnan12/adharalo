# 🎬 My FTP Stremio Addon v2

Admin panel সহ Personal Stremio Addon।

## 🚀 Render Deploy

1. GitHub-এ push করুন (node_modules বাদে)
2. render.com → New Web Service → আপনার repo
3. Environment Variables সেট করুন:
   - `ADMIN_PASSWORD` → আপনার পছন্দের password
   - `SESSION_SECRET` → যেকোনো random string
4. Deploy!

## 📲 Stremio Install

Deploy হলে: `https://your-app.onrender.com` → "Stremio Install" বাটন

## 🔐 Admin Panel

`https://your-app.onrender.com` খুলুন → password দিয়ে login করুন → মুভি/সিরিজ যোগ করুন।

Default password: `admin123` (Render-এ ADMIN_PASSWORD দিয়ে বদলান)
